# Troubleshooting Log

This file tracks process violations, errors, solutions, and recurring patterns during the development of the Aladia Real-Time ETL Pipeline.

## Format

Each entry should include:
- **Date/Time**: When the issue occurred
- **Error**: Description of the error or process violation
- **Root Cause**: Why it happened
- **Solution**: How it was resolved
- **Prevention**: How to avoid it in the future
